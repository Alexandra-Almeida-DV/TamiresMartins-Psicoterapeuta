# 🌿 Tamires Martins — Website Institucional

> Mais que um website institucional: uma plataforma estratégica desenhada para transmitir autoridade, acolhimento e facilitar a conversão de visitantes em pacientes.

---

## ✨ Sobre o Projeto

Este repositório contém o frontend de um website institucional de alta performance desenvolvido para a psicoterapeuta **Tamires Martins**.

O projeto foi concebido para unir:

- Sofisticação visual
- Acessibilidade
- Experiência fluida
- Performance
- Conversão estratégica

A proposta vai além de “ter presença online”. O objetivo foi construir uma experiência digital capaz de fortalecer a credibilidade profissional e reduzir fricções no processo de contato e agendamento.

---

# 🚀 Estratégia de Negócio Aplicada

Diferente de um site comum, este projeto foi estruturado sobre pilares que conectam tecnologia e resultado de negócio.

---

## 1️⃣ Autoridade e Credibilidade Visual

A identidade visual foi construída para transmitir:

- Acolhimento
- Segurança
- Sofisticação
- Profissionalismo

### Decisões Estratégicas

- Tipografia clássica com **Cormorant Garamond**
- Uso de **Glassmorphism** para profundidade visual elegante
- Paleta minimalista e tons suaves
- Ornamentos sutis para reforçar identidade premium

O resultado é uma experiência compatível com o nicho de saúde mental, fortalecendo confiança desde o primeiro acesso.

---

## 2️⃣ UX Focada em Conversão

Toda a navegação foi desenhada para reduzir atrito.

O paciente consegue:

- Entender rapidamente a proposta profissional
- Navegar sem distrações
- Entrar em contato em poucos cliques

### Recursos Implementados

- Drawer responsivo para mobile
- CTA estratégicos
- Fluxo de contato simplificado
- Integração com WhatsApp
- Hierarquia visual orientada à ação

---

## 3️⃣ SEO e Performance

O projeto foi otimizado para performance e indexação local.

### Estratégias Utilizadas

- Estrutura semântica HTML5
- Otimização de imagens
- Componentização performática
- Lazy loading
- Responsividade completa
- Boas práticas de acessibilidade

### Potencial de Ranqueamento

Preparado para buscas locais como:

- `"Psicóloga em Araraquara"`
- `"Psicoterapeuta online"`
- `"Atendimento psicológico Araraquara"`
